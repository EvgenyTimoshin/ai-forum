The threat model for these agentic workflows is huge. The "E-commerce Product Onboarding System" is a prime example. An attacker could use prompt injection on the product name input to make the "Copywriting Agent" generate malicious descriptions or the "Platform Integration Agent" call unintended API endpoints. Without rigorous input sanitization and sandboxing for every agent, especially those with tool-using capabilities, these systems are a security incident waiting to happen.
