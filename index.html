AI Blogs

This site hosts a collection of blogs generated by AI. It's a project to transform long-form content like PDFs and web articles into clean, readable web pages. AI agents automate content creation through several workflows: PDF conversion, web source distillation, and topic research.

View on GitHub
Agent Workflows

To stay on track, agents use checklists to manage their progress through each workflow.

PDF Conversion Workflow
  1. Convert PDF: Transforms the PDF file into a Markdown document for easier processing.
  2. Setup Webpage: Creates a new HTML page from a base template.
  3. Process & Condense: Reads the content section by section, creating a condensed version for the web.
  4. Finalize: Reviews the completed page for formatting and coherence.
Internet Source Workflow
  1. Gather Sources: Collects the user-provided URLs as the basis for the content.
  2. Setup Webpage: Creates a dedicated directory and HTML file for the new post.
  3. Synthesize Content: Reads the source articles, identifies key points, and writes a unified summary.
  4. Finalize: Reviews the generated blog post for quality and coherence.
Internet Search Workflow
  1. Understand & Search: Identifies the core topic and finds 3-5 high-quality sources online.
  2. Setup Webpage: Creates a dedicated directory and HTML file.
  3. Outline & Synthesize: Builds an outline from the sources and writes condensed content.
  4. Finalize & Cite: Reviews the final page and adds a list of the original sources.
